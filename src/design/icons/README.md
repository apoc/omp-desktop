# OMP Icon Pack

**58 icons · 8 categories**

Hand-drawn for the OMP Desktop agent canvas. 16×16 grid · 1.5 stroke · round caps/joins. Every icon carries a single filled accent dot — the signature — placing the eye where the action is.

## Usage

```tsx
import { Icon } from "./ui";

<Icon name="plan" size={14} color="var(--accent)" />
```

## Color tokens

Each icon's accent dot is recolorable. The pack ships expecting these tones to be available:

| Token | Hex | Used for |
|---|---|---|
| `--accent` | `#ff8a4c` | edit, write, plan-approval |
| `--cyan` | `#5ce7e0` | read |
| `--lilac` | `#b89cff` | search, thinking |
| `--amber` | `#ffc857` | bash, warn |
| `--lime` | `#9ee07a` | plan task, ok |
| `--magenta` | `#ff7ab8` | write |
| `--rose` | `#ff6b8a` | risk, error |

## Files

- `svg/*.svg` — 58 individual SVG files at 24px (recolor via CSS `color` for stroke; the dot is hard-coded orange — edit the inline `fill` to retune)
- `manifest.json` — full machine-readable spec including paths
- `icons.tsx` — drop-in React component identical to the in-app version
